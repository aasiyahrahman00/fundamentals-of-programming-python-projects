# Fundamentals of Programming Python Projects

This repository contains Python projects and formative activities completed as part of my Fundamentals of Programming module.

The projects were developed to practise core programming concepts including variables, control flow, functions, lists, dictionaries, input validation, debugging and problem-solving using Python.

---

## Contents

### Formative Activities

Short Python exercises used to practise:

- basic syntax
- input and output
- arithmetic operations
- functions
- loops and conditionals
- lists
- variable scope
- error handling

These activities helped build foundational programming confidence before progressing to larger projects.

---

### Financial Tracker

A Python command-line program that allows users to:

- add transactions
- delete transactions
- filter transactions
- calculate summaries
- track income and expenses

The program uses lists and dictionaries to organise transaction data clearly and efficiently.

#### Skills Practised

- functions
- loops
- dictionaries
- lists
- input validation
- debugging
- logical problem-solving

---

### Text Adventure Game: *The Bunny and the Golden Key*

A Python text-based adventure game where the player explores rooms, collects items, avoids traps and searches for the Golden Key before the burrow collapses.

The game includes:

- room navigation
- inventory tracking
- hidden traps
- hints
- move countdowns
- replay functionality
- user command handling

#### Skills Practised

- dictionaries and lists
- functions
- loops and conditionals
- game state management
- debugging and testing
- edge case handling

## Reflection

These projects helped strengthen my understanding of core programming concepts and showed me how smaller programming techniques combine to create larger applications.

Through debugging and testing, I became more confident identifying logic errors, improving code structure and handling invalid user input more effectively.

The text adventure game especially helped me understand how functions, lists, dictionaries and control flow can work together to manage game state and user interaction within a larger program.
